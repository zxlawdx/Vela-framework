Vela Framework • Python Desktop Runtime

Construa aplicações desktop modernas usando HTML, CSS e Python.

O Vela Framework foi criado para permitir que desenvolvedores Python construam aplicações desktop com uma arquitetura moderna, utilizando tecnologias web como HTML, TailwindCSS e JavaScript, mas sem depender de um navegador tradicional.

⚡

Runtime Desktop

Aplicações desktop reais usando pywebview, sem Electron e sem abrir navegador.

🎨

Interface Moderna

Crie interfaces usando HTML, TailwindCSS e JavaScript moderno.

🧠

Python First

Rotas, lógica, serviços, automações e integrações ficam centralizados no Python.

🧩

Arquitetura Modular

Sistema de apps, rotas, templates e layouts inspirado em frameworks modernos.

Como o Vela funciona

01

Backend Python

O Python controla rotas, lógica de negócio, integração com sistema operacional e comunicação com JS.

02

Shell Runtime

O shell do framework controla sidebar, topbar, layouts, loading e navegação SPA.

03

Frontend Livre

Cada rota renderiza apenas o conteúdo da página, permitindo total liberdade ao desenvolvedor.
